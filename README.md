* 줄거리 구글번역API
>-- 설치 과정
>'''
>pip install googletrans==4.0.0-rc1
>'''
>위에 코드로 아나콘다 프롬프트에 설치를 합니다
>
>+ 버전을 선택하는 이유는,
>pip install googletrans 만 입력해 설치하게되면 3.0 버전으로 설치되게 되고
>오류가 많아서 사용이 어렵습니다
>그러므로 4.0 버전으로 설치를 해야합니다
